在anaconda prompt中查看pytorch是否成功安装、查看版本号
 首先查看环境列表
conda env list
激活自己创建好的虚拟环境
conda activate 环境名
进入虚拟环境之后，输入python命令
import torch
 print(torch.__**version__**) #注意是双下划线
 结束：conda deactivate
删除环境conda remove -n env_name --all

创建环境：conda create --name tf2.2.0 python=3.7

激活环境：conda activate tf2.2.0

装cuda和cudnn ：conda install cudatoolkit=10.1.243 cudnn=7.6.5


cmd 退出python
