# 大模型训练教材总目录

> 这是一套面向工程实践的“大模型训练教科书式讲义”。
>
> 目标不是让你只会背概念，而是让你真正理解：大模型是怎么训练出来的、为什么后训练是业务团队最常做的工作、怎样设计一个能稳定上线的后训练流程，以及训练任务出问题后如何排查和运维。

---

## 这套教材适合谁

- 想系统学习大模型训练的同学
- 需要做业务模型微调、对齐、评测、上线的工程师
- 负责训练平台、GPU 集群、训练作业稳定性的运维/平台同学
- 想从“只会调用 API”走向“理解模型训练全链路”的开发者

---

## 你会学到什么

读完这套教材后，你应该能回答下面这些问题：

- 大模型训练到底分哪些阶段，预训练和后训练有什么本质区别
- 为什么绝大多数企业团队做的是后训练，而不是从零预训练
- SFT、LoRA、QLoRA、DPO、PPO、GRPO、KTO、ORPO 分别解决什么问题
- 后训练的数据应该怎么构造，哪些数据看起来很多但其实对效果有害
- loss 降了为什么模型效果还是不好
- 为什么训练成功不等于业务可上线
- 一次后训练任务从数据准备到上线回滚，运维到底要盯哪些指标、做哪些预案

## 大模型训练全景图

```text
数据准备 -> 分词与模板化 -> 训练任务启动 -> 前向计算 -> loss 计算
        -> 反向传播 -> 参数更新 -> checkpoint 保存 -> 离线评测
        -> 人工评审 -> 灰度验证 -> 上线部署 -> 监控回收 -> 下一轮迭代
```

更完整的产业链视角如下：

```text
阶段 1：预训练
  目标：让模型学会语言规律、世界知识、代码模式、常识结构
  结果：得到 Base 模型

阶段 2：监督后训练（SFT / CPT / PEFT）
  目标：让模型学会按要求说话、按格式输出、学会业务任务
  结果：得到 Instruct / Domain 模型

阶段 3：偏好对齐（DPO / PPO / GRPO / KTO / ORPO / SimPO）
  目标：让模型的回答更符合“好回答”的标准
  结果：得到更稳定、更可控、更符合业务偏好的模型

阶段 4：评测、部署、运维
  目标：让训练成果真的变成可用服务
  结果：业务上线、监控闭环、持续迭代
```

## 后训练为什么是重点

因为现实里绝大多数团队没有能力也没有必要从零预训练一个模型，但几乎所有做大模型业务落地的团队都离不开后训练。

你可以把它理解为：

- 预训练是在“造脑子”
- 后训练是在“教这个脑子怎么为你的业务做事”

后训练之所以重要，是因为业务真正关心的问题通常都发生在后训练阶段：

- 怎么让模型学会公司自己的知识和表达风格
- 怎么让模型输出固定格式，比如 JSON、SQL、代码补全模板
- 怎么提升安全性、拒答策略、长答案稳定性、工具调用习惯
- 怎么让模型在数学、代码、问答、客服、企业知识库等任务上表现更好

## 学习时要抓住的主线

学习大模型训练，最容易犯的错误是“见方法就学，见名词就背”，最后只会堆术语，不会做判断。

真正应该抓住的主线只有四条：

1. 训练目标是什么
2. 数据长什么样
3. 训练时到底更新了哪些参数
4. 训练结束后如何证明模型真的变好了

无论是 SFT、LoRA、DPO 还是 PPO，本质上都逃不开这四个问题。

## 一套最实用的工程认知

如果你只记住一句话，请记住下面这句：

> 大模型训练不是“跑一个脚本”，而是“数据、模型、资源、评测、运维共同组成的生产系统”。

所以你读后面的内容时，请始终带着下面这五个视角：

- 算法视角：方法原理对不对
- 数据视角：数据好不好
- 工程视角：能不能稳定跑完
- 评测视角：效果是否真的提升
- 运维视角：出了问题能不能快速恢复

## 建议的实践节奏

### 第 1 阶段：认知建立

- 看懂训练全景图
- 搞清楚预训练、SFT、偏好对齐的关系
- 明白什么叫 loss，什么叫 checkpoint，什么叫评测

### 第 2 阶段：跑通第一次后训练

- 选择一个 7B 左右的基础或指令模型
- 用一个小型 SFT 数据集做 LoRA 微调
- 学会看日志、看 loss、看 checkpoint

### 第 3 阶段：做质量优化

- 改数据质量
- 改数据配比
- 调学习率、batch size、epoch、max length
- 做离线评测和人工评测

### 第 4 阶段：做对齐与生产化

- 尝试偏好数据
- 建立标准评测集
- 接入训练监控、告警、回滚和上线流程

## 最后提醒

这套教材会反复强调一个事实：

> 后训练里，决定上限的通常不是“你会不会一个很新的算法名词”，而是“你是否把数据、目标、评测和运维闭环做完整了”。

如果你准备开始正式学习，请从第 1 册开始读；如果你现在就在做业务微调，请直接进入第 2 册和第 6 册。
